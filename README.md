An advanced prompt engineering framework and testing harness engineered to visually monitor automated learning UIs, enforce rigid instructional constraints, mitigate LLM persona drift, and output structured JSON/Markdown metadata.

---

## 🏗️ Core Engineering Solutions
This framework was built to overcome severe hardware and multimodal token limits inherent to real-time audio/video streaming APIs.

* **Micro-Session Protocol:** Mitigates aggressive token limits and memory fragmentation on live streaming endpoints by enforcing 15-minute context-clearing sprints.
* **Visual Anchor Detection:** Dynamically tracks UI geometry to toggle system behaviors between Live and Retrospective modes, successfully eliminating AI persona drift.
* **Context Injection Bypass:** Routes massive visual token bloat from long-form video media by utilizing manual fast-forwarding and voice-based data injections.
